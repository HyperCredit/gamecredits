What is Gamerscoin?

Gamers-Coin is a peer-to-peer Internet currency that enables instant payments to anyone in the gaming world.

Gamerscoin is a lite version of Bitcoin using scrypt as a proof-of-work algorithm. Used Litecoin as Basic Source Code.

2.5 minute block targets
subsidy halves in 840k blocks (~4 years)
~84 million total coins

The rest is the same as Bitcoin.

50 coins per block
2016 blocks to retarget difficulty
